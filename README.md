# Run Tracker IA 🏃

Application web d'analyse et prédiction de performance en course à pied.

## Ce que ça fait
- Visualise la progression de l'allure sur toutes les courses
- Prédit le temps au semi-marathon via régression linéaire
- Permet d'ajouter des courses dynamiquement

## Technologies
Python · Flask · Pandas · Matplotlib · HTML/CSS · JavaScript

## Lancer le projet
pip install flask
python app.py
Ouvrir localhost:5000
